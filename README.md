# yii2-lib
Simple user registration and authentication module for Yii2

## Now in development! ## 

## Instalation: ##

add git repository to project composer.json

```
    "repositories":[
        {
            "type":"git",
            "url":"https://github.com/svsoft/yii2-lib"
        }
    ],
```
And
```
composer require svsoft/yii2-lib dev-master
```

## Composition: ##

FileBehavior - Makes upload adn display file for model attributes
